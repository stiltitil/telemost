# Telemost Record Reminder

Расширение для Google Chrome, которое показывает заметное напоминание о записи после входа во встречу Яндекс Телемоста.

## Что делает

- ждет момента, когда вы уже подключились к звонку;
- показывает всплывающую плашку `Поставь запись`;
- проговаривает голосом `ПОСТАВЬ ЗАПИСЬ`;
- повторяет голосовое напоминание каждые 18 секунд, пока плашка не закрыта;
- закрывает плашку по галочке `Запись включена, можно закрыть это окно`.

## Как установить

1. Откройте `chrome://extensions/`
2. Включите `Режим разработчика`
3. Нажмите `Загрузить распакованное расширение`
4. Выберите папку `chrome-telemost-record-reminder`

## Где работает

- `https://telemost.360.yandex.ru/j/*`
- `https://telemost.yandex.ru/j/*`
